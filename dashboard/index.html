                                        Вход

                                        Чтение дашборда — свободное. Авторизация нужна для действий: постановка дорожной карты, запуск симулятора, генерация формулировок, запуск сбора данных.

                                        О приложении

                                        Городской Разум

                                        Платформа прогнозного управления городом на основе 9-элементной модели Мейстера. Читает потоки новостей и обращений, объясняет «почему», прогнозирует «что будет», предлагает «что сделать».

                                        Мониторинг

                                        • Состояние города — красная полоса сверху = критичные сигналы, жёлтая = наблюдение.
                                        • Ключевые показатели СБ / ТФ / УБ / ЧВ — 4 вектора Мейстера на шкале 1–6. Клик по плитке → разбор «откуда эта цифра».
                                        • Погода + индекс комфорта, Доверие + Счастье — социальный срез.
                                        • Активные петли — замкнутые причинно-следственные цепочки, подлежащие «разрыву».

                                        Аналитика

                                        • Модель Мейстера — 9 элементов + связи (Cytoscape). Клик на узле → легенда + кнопка «Почему?».
                                        • Что если… (симулятор бабочки) — поверните ручку, получите каскад по графу.
                                        • Репутация в медиа — топ-критики, вирусные посты, тренд негатива за 7 дней.
                                        • Топики — что обсуждают прямо сейчас, тренд vs прошлая неделя.
                                        • Бенчмарк 6 городов — ранг вашего города на фоне пилотов.

                                        Прогноз и планирование

                                        • Прогноз 3 мес. — алгоритм Мейстера, краткая сводка.
                                        • Прогноз 7/30/90 дней с интервалом — Holt-метод + доверительная полоса.
                                        • Форсайт 5 лет — 3 сценария + 10 мегатрендов.
                                        • Бюджет — рекомендованное распределение по векторам.
                                        • Дорожная карта — квартальные вехи для закрытия разрыва по вектору.

                                        Действия

                                        • Поручения — приоритизированный чек-лист, собранный из повестки + кризисов.
                                        • Симулятор решений — 10 типовых ходов, 3 сценария, стоимость, риски.
                                        • Кейсы-аналоги — 15 лучших практик из городов РФ, подобранных под ваши слабые векторы.
                                        • Рыночные ниши — куда звать бизнес на основе жалоб горожан.
                                        • Готовые формулировки — AI даёт 3 варианта заявления: формальный / эмпатичный / мобилизующий.

                                        Как использовать

                                        • Переключение города — кнопка с эмодзи 🏰 в шапке.
                                        • Автообновление — каждые 10 минут; состояние системы — каждые 60 сек.
                                        • Состояние системы — кнопка справа в шапке: БД / Redis / DeepSeek / планировщик.
                                        • Справка — эта модалка, кнопка «?» слева от часов.
                                        Под капотом: FastAPI + asyncpg + DeepSeek AI. Данные собираются из Telegram, ВКонтакте, RSS и обогащаются LLM. Источники документированы в docs/NEWS_SOURCES.md. Весь код открыт, без sklearn / tensorflow — адаптеры лёгкие, прозрачные, тестируемые.
