# 🏙️ Городской Разум

[![tests](https://github.com/Dron939-sketch/-/actions/workflows/tests.yml/badge.svg)](https://github.com/Dron939-sketch/-/actions/workflows/tests.yml)

> *«Город — это механизм из 9 шестерёнок. Если одна заедает — страдают все. Обычные отчёты показывают только температуру механизма. Алгоритм Мейстера показывает, КАК сцеплены шестерёнки, ГДЕ трение, и КАКУЮ шестерёнку повернуть, чтобы всё заработало.»*

**«Городской Разум»** — первая в России платформа, которая видит город не как набор статистических показателей, а как **живую систему**, подчиняющуюся законам обратных связей.

В её основе — **алгоритм Андрея Мейстера**, создателя уникальной методологии анализа закрытых систем. Впервые эта технология адаптирована для городского управления: она выявляет скрытые петли, которые годами держат город в одном состоянии, показывает, почему проблемы возвращаются, и даёт мэру готовый план их разрыва.

Платформа собирает новости, соцсети и обращения граждан, прогоняет их через AI-обогатитель, рассчитывает 4 ключевых вектора (**СБ** / **ТФ** / **УБ** / **ЧВ**), формирует ежедневную повестку и дорожную карту для главы города.

## 📸 Андрей Мейстер — автор алгоритма

![Андрей Мейстер](andrey-meister.jpg)

**Андрей Мейстер** — создатель уникальной методологии анализа закрытых систем. Его алгоритм позволяет:

| Возможность | Описание |
|-------------|----------|
| 🔍 **Выявлять скрытые петли обратной связи** | Механизмы, которые годами держат город в одном состоянии |
| 🎯 **Находить первопричины** | Не лечить симптомы, а устранять корень проблемы |
| 🦋 **Прогнозировать эффект бабочки** | Видеть, как изменение одного параметра повлияет на всю систему |
| ⚡ **Определять точки разрыва** | Находить, какое действие даст максимальный эффект |

*«Алгоритм Мейстера — это рентгеновский снимок городской системы. Видит то, что не видно на поверхности.»*

## 🚀 Быстрый старт

### Требования

- Python 3.11+
- PostgreSQL 15+
- Redis 7+
- Docker (опционально)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/Dron939-sketch/gorodskoy-razum.git
cd gorodskoy-razum

# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Заполните:
# - DEEPSEEK_API_KEY
# - OPENWEATHER_API_KEY
# - VK_API_TOKEN
# - TELEGRAM_API_ID / TELEGRAM_API_HASH

# Запуск базы данных и Redis
docker compose up -d db redis

# Инициализация базы данных
psql "postgresql://citymind:citymind@localhost:5432/citymind" -f migrations/init_db.sql

# Запуск приложения
uvicorn api.main:app --reload
